# 🌍 Ready9ja - Global Marketplace Platform

Follow this link to view the progress https://vendeer-3fc02.web.app/ 

**Ready9ja** is a scalable, mobile-friendly e-commerce platform that connects vendors and buyers in a seamless digital marketplace. Built with **React.js** (frontend) and **Firebase** (backend), the platform offers secure authentication, real-time product management, and a smooth user experience.

---

## 🚀 Features

- 🔐 User Authentication (Firebase Auth)
- 🛍️ Vendor Dashboard for Product Uploads
- 🧾 Cart & Order Management
- 🔎 Search & Filter Products
- 📊 Real-time Inventory Tracking
- 🌐 Responsive UI (Mobile & Web)
- 🔔 Notifications via Firebase Cloud Messaging (FCM)
- 💬 Integrated Support Chat (via Firebase Firestore)

---

## 🛠️ Tech Stack

| Frontend | Backend | Database | Hosting |
|----------|---------|----------|---------|
| React.js | Firebase Auth | Firestore | Firebase Hosting |
| TailwindCSS | Firebase Functions | Firebase Storage | Vercel (optional) |
